Big Data Analytics

In the "Big Data Analytics" subject, I explored various tools and platforms for processing and analyzing large-scale datasets.
Through hands-on assignments, I learned how to harness the power of Google BigQuery, Azure Data Lake, Azure SQL, Azure Synapse, and Azure Data Factory (ADF) to extract, transform, and visualize data using Power BI.






Assignment

BigQuery Google

I worked with Google BigQuery, a serverless data warehouse, to query and analyze large datasets efficiently. I learned to optimize queries and leverage the scalability of BigQuery for quick data processing.


Azure Data Lake to Power BI

I explored Azure Data Lake, a cloud-based data storage and analytics solution, and connected it to Power BI for data visualization. This integration allowed me to gain insights from vast amounts of unstructured data.


Azure SQL to Power BI

I utilized Azure SQL, a fully managed relational database service, to store and manage structured data. I integrated Azure SQL with Power BI for real-time data analysis and reporting.


Azure Synapse to External Table

I worked with Azure Synapse, a limitless analytics service, to create external tables and access data from various data sources. This approach allowed me to perform unified analytics across diverse datasets.


ADF to Azure SQL to Azure Data Lake

I leveraged Azure Data Factory (ADF) to orchestrate data movement and transformation between Azure SQL and Azure Data Lake. ADF facilitated the seamless flow of data for analysis and processing.


Outcomes

- Developed proficiency in using Google BigQuery for scalable and efficient data analysis.

- Gained expertise in connecting Azure Data Lake, Azure SQL, and Azure Synapse to Power BI for comprehensive data visualization and insights.

- Acquired the ability to orchestrate data movement and transformation with Azure Data Factory between different data sources.

- Mastered the integration of various cloud-based data storage and analytics platforms to enable seamless big data analytics.
