Web and Social Media Analytics

In the "Web and Social Media Analytics" subject, I delved into the exciting world of web scraping, sentiment analysis, and network analysis.
Through a series of assignments, I explored various techniques and tools to gather data from web sources, analyze sentiments, and visualize networks for valuable insights.






Assignment

Web Scraping Using Python

I learned web scraping techniques using Python to extract data from websites such as Swiggy, IMDb, and others. I utilized libraries like Beautiful Soup and Scrapy to navigate web pages, scrape information, and store it for analysis.


Sentiment Analysis

I explored sentiment analysis, using natural language processing (NLP) techniques to determine the sentiments expressed in text data. I performed sentiment analysis on social media platforms like Twitter and Facebook using Graph API, as well as on other web sources.


Google Sheet Add-ons for Data Analysis

I utilized Google Sheet add-ons to perform data analysis and manipulate data collected through web scraping. These add-ons facilitated quick and efficient data processing and visualization.


NLP Techniques

I applied various NLP techniques such as lemmatization, segmentation, part-of-speech tagging, and stemming to preprocess textual data for sentiment analysis and other analyses.


Sentiment Analysis Using ChatGPT

I explored generic sentiment analysis using ChatGPT, an advanced language model. This allowed me to understand how state-of-the-art language models can interpret and classify sentiments in text data.


Network Analysis Using Gephi

I visualized social networks and relationships by creating network diagrams using Gephi. This helped me understand the connections between entities and identify influential nodes.


Outcomes

- Developed proficiency in web scraping using Python, enabling data collection from various web sources.

- Acquired expertise in sentiment analysis to understand and analyze sentiments expressed in textual data.

- Gained the ability to use Google Sheet add-ons for efficient data analysis and visualization.

- Mastered NLP techniques for preprocessing textual data, improving the accuracy of sentiment analysis.

- Successfully visualized networks using Gephi, allowing me to identify key relationships and influential nodes.
